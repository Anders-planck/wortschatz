Direction B
Tinte

Preposition — auf
9:4187%
⌕
WECHSELPRÄPOSITION
auf
su, sopra
♪
Casi
+ AKK movimento → dove?
„Leg es auf den Tisch."
Mettilo sul tavolo.
+ DAT stato → dove?
„Es liegt auf dem Tisch."
È sul tavolo.
Quando si usa
auf = su una superficie. Akkusativ quando c'è movimento verso la superficie. Dativ quando qualcosa è già lì. Confronta: an (a contatto verticale), über (sopra senza contatto).
